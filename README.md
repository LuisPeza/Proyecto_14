# Proyecto_14
Ultimo proyecto de Análisis de datos
<h1>Telefonía Virtual</h1>

<h2>Identificación del problema</h2>
Identificar a los operadores menos eficientes En la industria del Call Center los operadores asumen un gasto operativo aproximadamente del 78 % de los gastos. Es por esto que es de suma importancia tener a los operadores mas eficientes. Con este nuevo desarrollo se ayudara a mitigar e identificar a los operadores deficientes para poder suplirlos o capacitarlos de mejor manera, con una mejor eficiencia en los operadores puede ayudar a mejorar la rentabilidad del negocio.

<h3>Objetivo</h3>
Identificar a operadores ineficientes, basado en 3 comportamientos.

- Si tiene un numero de llamadas salienten reducido
- Si tiene un numero de llamadas perdidas 
- Si el tiempo de espera en contestar las llamadas es prolongado.
